## 章节说明

本章节介绍PostgreSQL 18的自动化性能调优功能，包括：

- **PostgreSQL 18性能优化新特性**：异步I/O、NUMA架构支持、并行查询追踪等
- **自动参数调优**：基于工作负载自动调整数据库参数
- **自动索引优化**：自动识别和创建缺失索引
- **自动统计信息更新**：根据数据变化率自动更新统计信息
- **自动VACUUM优化**：智能调整VACUUM策略（支持并行VACUUM）

### PostgreSQL 18核心新特性

本章节重点介绍了PostgreSQL 18引入的核心性能优化特性：

1. **异步I/O支持**（2.1）：通过异步I/O基础设施提升并发性能，支持worker和io_uring两种方法
2. **NUMA架构支持**（2.2）：优化多处理器系统的内存访问性能
3. **并行查询追踪**（2.3）：pg_stat_statements新增并行工作进程追踪列
4. **EXPLAIN增强**（2.4）：提供即时性能诊断和优化建议

### 快速开始

1. **启用异步I/O**（推荐优先配置）
   - 阅读：[2.1 异步I/O支持](./01-异步I-O支持.md)
   - 配置：`io_method = 'worker'`（或 `'io_uring'`）

2. **配置自动索引优化**
   - 阅读：[2.6 自动索引优化](./06-自动索引优化.md)
   - 运行自动索引检测函数

3. **启用自动参数调优**
   - 阅读：[2.5 自动参数调优](./05-自动参数调优.md)
   - 配置自动参数调优函数
