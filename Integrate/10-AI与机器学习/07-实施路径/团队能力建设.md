## 一、概述

PostgreSQL AI应用的团队能力建设指南，包括核心技能要求、培训计划、学习资源、能力评估等，帮助团队快速掌握PostgreSQL AI技术栈。

## 二、能力模型

### 2.1 核心技能要求

**基础技能**:

- PostgreSQL基础（SQL、索引、事务）
- 向量检索原理（相似度计算、索引算法）
- 基础AI概念（Embedding、LLM）

**进阶技能**:

- pgvector高级特性（HNSW、IVFFlat）
- pgai使用（Vectorizer、LLM调用）
- 性能优化（查询优化、索引调优）

**高级技能**:

- PostgresML模型训练
- 混合查询优化
- 大规模系统设计

### 2.2 角色分工

| 角色 | 核心技能 | 培训时间 |
|------|:--------|:--------|
| **DBA** | PostgreSQL管理、性能优化 | 2-4周 |
| **开发工程师** | SQL、pgvector、pgai | 1-2周 |
| **数据工程师** | ETL、数据治理 | 1-2周 |
| **AI工程师** | LLM、向量检索 | 1周 |
| **架构师** | 系统设计、技术选型 | 2-3周 |

### 2.3 技能矩阵

| 技能 | 初级 | 中级 | 高级 |
|------|:----:|:----:|:----:|
| **PostgreSQL基础** | ✅ | ✅ | ✅ |
| **pgvector使用** | ✅ | ✅ | ✅ |
| **pgai使用** | ✅ | ✅ | ✅ |
| **性能优化** | ⚠️ | ✅ | ✅ |
| **PostgresML** | ❌ | ⚠️ | ✅ |
| **系统设计** | ❌ | ⚠️ | ✅ |

## 三、培训计划

### 3.1 基础培训

**时间**: 1-2周

**内容**:

1. PostgreSQL基础（3天）
   - SQL基础
   - 索引和查询优化
   - 事务和并发控制

2. pgvector基础（2天）
   - 向量数据类型
   - 相似度操作符
   - HNSW索引创建

3. pgai基础（2天）
   - embedding()函数
   - chat_complete()函数
   - Vectorizer使用

**培训材料**:

- [向量处理能力文档](../03-核心能力/向量处理能力-pgvector.md)
- [AI原生调用文档](../03-核心能力/AI原生调用-pgai.md)

### 3.2 进阶培训

**时间**: 2-3周

**内容**:

1. 性能优化（1周）
   - 索引优化策略
   - 查询计划分析
   - 系统参数调优

2. 混合查询（1周）
   - 向量+SQL联合查询
   - 向量+全文搜索
   - 多模态查询

3. 应用场景（1周）
   - RAG系统设计
   - 推荐系统实现
   - 风控系统设计

**培训材料**:

- [性能优化技术文档](../03-核心能力/性能优化技术.md)
- [混合查询能力文档](../03-核心能力/混合查询能力.md)
- [应用场景文档](../04-应用场景/)

### 3.3 专项培训

**时间**: 1-2周

**内容**:

1. PostgresML（3天）
   - 模型训练
   - 模型推理
   - 特征工程

2. 大规模部署（3天）
   - Citus分布式
   - 高可用架构
   - 监控告警

3. 最佳实践（2天）
   - 代码规范
   - 性能调优
   - 故障排查

## 五、能力评估

### 5.1 技能评估

**评估标准**:

- **初级**: 能够使用基本功能
- **中级**: 能够优化和调优
- **高级**: 能够设计和架构

**评估方式**:

```sql
-- 实践任务评估
-- 任务1：创建向量索引
CREATE INDEX ON documents
USING hnsw (embedding vector_cosine_ops);

-- 任务2：优化查询
EXPLAIN ANALYZE
SELECT * FROM documents
WHERE embedding <=> query_vector < 0.8;

-- 任务3：实现RAG系统
CREATE FUNCTION rag_query(...);
```

### 5.2 认证考试

**PostgreSQL认证**:

- PostgreSQL Professional Certification
- pgvector专项认证（如提供）

**评估内容**:

- 理论知识（30%）
- 实践操作（50%）
- 案例分析（20%）
