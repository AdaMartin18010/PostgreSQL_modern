## 🎯 核心主题

### 基本操作

- **选择（Selection）** - σ操作，对应WHERE子句
- **投影（Projection）** - π操作，对应SELECT列列表
- **并（Union）** - ∪操作，对应UNION操作符
- **差（Difference）** - −操作，对应EXCEPT操作符
- **笛卡尔积（Cartesian Product）** - ×操作，对应CROSS JOIN

### 导出操作

- **连接（Join）** - ⋈操作，包括等值连接和自然连接
- **交（Intersection）** - ∩操作，对应INTERSECT操作符
- **除（Division）** - ÷操作，用于表达"所有"查询

### 代数性质

- **交换律** - 选择和连接的交换律
- **结合律** - 选择和连接的结合律
- **分配律** - 选择对并和差的分配律
- **幂等律** - 选择和投影的幂等律

### 优化规则

- **选择下推** - 尽可能早地执行选择操作
- **投影下推** - 尽可能早地执行投影操作
- **连接顺序优化** - 选择最优的连接顺序
