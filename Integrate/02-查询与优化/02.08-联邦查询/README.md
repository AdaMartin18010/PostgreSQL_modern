## 📊 联邦查询分类

### FDW类型

- postgres_fdw - PostgreSQL到PostgreSQL
- mysql_fdw - MySQL数据源
- oracle_fdw - Oracle数据源
- file_fdw - 文件数据源
- multicorn - Python FDW框架

### 应用场景

- 数据集成
- 跨数据库查询
- 数据迁移
- 数据同步
