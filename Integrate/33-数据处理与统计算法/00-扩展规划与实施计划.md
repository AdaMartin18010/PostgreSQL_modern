## 🆕 新增目录规划

### 14-高维数据分析

**目标**: 处理高维数据的降维、可视化和分析

**文档列表**:

1. 主成分分析（PCA）- 详细数学推导、特征值分解、方差贡献率
2. 线性判别分析（LDA）- Fisher准则、类间散度、类内散度
3. 独立成分分析（ICA）- 盲源分离、非高斯性
4. 张量分解 - CP分解、Tucker分解、多维数据分析
5. 非线性降维 - t-SNE、UMAP、流形学习
6. 因子分析 - 探索性因子分析、验证性因子分析

### 15-数值线性代数

**目标**: 矩阵运算、线性系统求解、数值计算

**文档列表**:

1. 矩阵分解 - LU分解、QR分解、Cholesky分解、SVD
2. 特征值与特征向量 - 幂法、QR算法、Lanczos方法
3. 线性系统求解 - 高斯消元、迭代法、共轭梯度法
4. 矩阵运算优化 - 稀疏矩阵、分块矩阵、并行计算
5. 数值稳定性分析 - 条件数、舍入误差、数值精度

### 16-统计推断

**目标**: 假设检验、置信区间、贝叶斯统计

**文档列表**:

1. 假设检验 - t检验、卡方检验、ANOVA、Mann-Whitney U检验
2. 置信区间 - 正态分布、t分布、Bootstrap方法
3. 贝叶斯统计 - 先验分布、后验分布、贝叶斯推断
4. 非参数统计 - 秩检验、符号检验、Kolmogorov-Smirnov检验
5. 多重比较 - Bonferroni校正、FDR控制、Holm方法
6. 功效分析 - 样本量计算、检验功效、效应量

### 17-高级机器学习

**目标**: 高级ML算法、深度学习基础、集成学习

**文档列表**:

1. 支持向量机（SVM） - 最大间隔、核函数、软间隔
2. 随机森林 - 决策树集成、特征重要性、OOB误差
3. 梯度提升 - XGBoost、LightGBM、AdaBoost
4. 神经网络基础 - 感知机、反向传播、激活函数
5. 集成学习 - Bagging、Boosting、Stacking
6. 强化学习基础 - Q-learning、策略梯度、价值函数

### 18-优化算法

**目标**: 线性规划、非线性优化、约束优化

**文档列表**:

1. 线性规划 - 单纯形法、对偶理论、整数规划
2. 非线性优化 - 梯度下降、牛顿法、拟牛顿法
3. 约束优化 - 拉格朗日乘数法、KKT条件、罚函数法
4. 全局优化 - 模拟退火、遗传算法、粒子群优化
5. 凸优化 - 凸集、凸函数、对偶问题

### 19-数据可视化

**目标**: SQL生成可视化数据、统计图表

**文档列表**:

1. 统计图表生成 - 直方图、箱线图、散点图数据
2. 时间序列可视化 - 趋势图、季节性分解
3. 多维数据可视化 - 热力图、雷达图、平行坐标
4. 交互式可视化 - 动态查询、实时更新

### 20-数据安全与隐私

**目标**: 数据加密、匿名化、差分隐私

**文档列表**:

1. 数据加密 - 对称加密、非对称加密、哈希函数
2. 数据匿名化 - k-匿名、l-多样性、t-接近性
3. 差分隐私 - ε-差分隐私、拉普拉斯机制、高斯机制
4. 安全聚合 - 安全多方计算、同态加密基础

### 21-数据流处理

**目标**: 实时计算、滑动窗口、流式聚合

**文档列表**:

1. 流式聚合 - 实时统计、滑动窗口聚合
2. 流式连接 - 流-流连接、流-表连接
3. 时间窗口 - 滚动窗口、滑动窗口、会话窗口
4. 流式异常检测 - 实时异常检测、概念漂移

### 22-信号处理

**目标**: 傅里叶变换、滤波、频谱分析

**文档列表**:

1. 傅里叶变换 - DFT、FFT、频域分析
2. 数字滤波 - 低通滤波、高通滤波、带通滤波
3. 频谱分析 - 功率谱密度、自相关函数
4. 小波变换 - 连续小波、离散小波、多分辨率分析

### 23-图论高级算法

**目标**: 扩展图算法深度

**文档列表**:

1. 最小生成树 - Kruskal、Prim算法
2. 网络流 - 最大流、最小割、Ford-Fulkerson
3. 图匹配 - 二分图匹配、匈牙利算法
4. 社区检测 - Louvain算法、模块度优化
5. 图嵌入 - Node2Vec、Graph2Vec、图神经网络基础

### 24-时间序列高级分析

**目标**: 深化时间序列分析

**文档列表**:

1. ARIMA模型 - 自回归、差分、移动平均、参数估计
2. 状态空间模型 - Kalman滤波、状态估计
3. 季节性分解 - STL分解、X-13ARIMA-SEATS
4. 协整分析 - Engle-Granger检验、Johansen检验
5. GARCH模型 - 波动率建模、条件异方差

### 25-关联规则挖掘

**目标**: 市场篮分析、频繁项集

**文档列表**:

1. Apriori算法 - 支持度、置信度、频繁项集
2. FP-Growth算法 - FP树构建、模式增长
3. 关联规则评估 - 提升度、卡方检验、规则剪枝
4. 序列模式挖掘 - GSP算法、PrefixSpan
