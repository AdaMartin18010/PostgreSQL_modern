## 📊 数据仓库分类

### 数据模型

- 星型模式（Star Schema）
- 雪花型模式（Snowflake Schema）
- 事实表（Fact Table）
- 维度表（Dimension Table）

### OLAP操作

- 上卷（Roll-up）
- 下钻（Drill-down）
- 切片（Slice）
- 切块（Dice）
- 旋转（Pivot）

### ETL流程

- 提取（Extract）
- 转换（Transform）
- 加载（Load）
- 增量ETL
- 全量ETL
