## 1. 概述

PostgreSQL欢迎社区贡献，有多种贡献方式。

**贡献类型**:

- 代码贡献
- 文档贡献
- 社区支持
- 资金支持

---

## 2. 贡献方式

### 2.1 代码贡献

```text
1. Fork项目
2. 创建分支
3. 编写代码
4. 提交PR
5. 代码审查
```

### 2.2 文档贡献

```text
1. 发现文档问题
2. 提交文档修复
3. 添加新文档
4. 翻译文档
```

---

## 3. 代码贡献

### 3.1 贡献流程

```bash
# 1. Fork项目
git clone https://github.com/postgres/postgres.git

# 2. 创建分支
git checkout -b feature/my-feature

# 3. 编写代码
# ... 编写代码 ...

# 4. 提交代码
git commit -m "Add new feature"

# 5. 推送分支
git push origin feature/my-feature

# 6. 创建PR
```

### 3.2 代码规范

```text
- 遵循PostgreSQL代码风格
- 编写测试用例
- 更新文档
- 通过代码审查
```
