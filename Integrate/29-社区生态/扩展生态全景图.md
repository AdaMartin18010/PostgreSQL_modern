## 1. 概述

PostgreSQL拥有丰富的扩展生态系统，覆盖各种应用场景。

**扩展类型**:

- 核心扩展
- 社区扩展
- 商业扩展
- 自定义扩展

---

## 2. 扩展分类

### 2.1 按功能分类

| 类别 | 扩展示例 |
|------|---------|
| **数据模型** | PostGIS、pgvector、hstore |
| **性能优化** | pg_stat_statements、pg_prewarm |
| **高可用** | Patroni、repmgr |
| **监控诊断** | pg_stat_monitor、pgBadger |

### 2.2 按来源分类

```text
- 官方扩展：PostgreSQL官方维护
- 社区扩展：社区维护
- 商业扩展：商业公司提供
```

---

## 3. 核心扩展

### 3.1 官方扩展

- **PostGIS** - 空间数据
- **pg_trgm** - 文本搜索
- **uuid-ossp** - UUID生成
- **hstore** - 键值存储

### 3.2 常用扩展

```sql
-- 查看已安装扩展
SELECT * FROM pg_extension;

-- 安装扩展
CREATE EXTENSION IF NOT EXISTS postgis;
```

---

## 4. 社区扩展

### 4.1 热门扩展

- **pgvector** - 向量搜索
- **TimescaleDB** - 时序数据
- **Apache AGE** - 图数据库
- **Citus** - 分布式数据库

### 4.2 扩展生态

```text
- pgxn.org - 扩展仓库
- GitHub - 扩展源码
- 社区维护 - 社区支持
```
