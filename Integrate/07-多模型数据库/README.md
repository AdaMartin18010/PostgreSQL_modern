```mermaid
flowchart TD
    A[需要选择数据模型?] --> B{数据类型?}
    B -->|结构化关系数据| C[关系表]
    B -->|半结构化文档| D[JSONB]
    B -->|向量相似度搜索| E[向量类型 + pgvector]
    B -->|图关系查询| F[图数据库 + Apache AGE]
    B -->|时间序列| G[时序数据库 + TimescaleDB]
    B -->|地理位置| H[空间数据 + PostGIS]
    B -->|混合需求| I[多模型集成]

    C --> J[优势: ACID、SQL标准<br/>适用: 传统业务数据]
    D --> K[优势: 灵活、嵌套结构<br/>适用: 文档存储、配置]
    E --> L[优势: 相似度搜索<br/>适用: AI应用、推荐系统]
    F --> M[优势: 关系查询<br/>适用: 社交网络、知识图谱]
    G --> N[优势: 时序优化<br/>适用: IoT、监控]
    H --> O[优势: 空间查询<br/>适用: 地理信息、LBS]
    I --> P[优势: 统一管理<br/>适用: 复杂业务场景]
```

| **图数据** | ✅ 扩展 | Apache AGE | ⭐⭐⭐⭐ | ⭐⭐⭐ | 社交网络、知识图谱 |
| **时序数据** | ✅ 扩展 | TimescaleDB | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | IoT、监控、分析 |
| **空间数据** | ✅ 扩展 | PostGIS | ⭐⭐⭐⭐ | ⭐⭐⭐ | 地理信息、LBS |
