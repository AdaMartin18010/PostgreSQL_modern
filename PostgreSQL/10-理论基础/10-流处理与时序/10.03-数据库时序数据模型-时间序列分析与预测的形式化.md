## 4. 实际应用

### 4.1 PostgreSQL时序支持

**使用TimescaleDB**：

```sql
-- 创建时序表
CREATE TABLE sensor_data (
    time TIMESTAMPTZ NOT NULL,
    sensor_id INTEGER,
    value DOUBLE PRECISION
);

-- 转换为超表
SELECT create_hypertable('sensor_data', 'time');

-- 时间序列查询
SELECT time_bucket('1 hour', time) AS hour,
       AVG(value) AS avg_value
FROM sensor_data
GROUP BY hour;
```
