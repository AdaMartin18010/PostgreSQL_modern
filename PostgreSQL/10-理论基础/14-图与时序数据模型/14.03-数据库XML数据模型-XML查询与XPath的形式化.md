## 2. 核心内容

### 2.1 XML数据模型

**XML文档结构**：

```haskell
-- XML文档
data XMLDocument = XMLDocument {
    root :: XMLElement,
    namespace :: Maybe Namespace
}

-- XML元素
data XMLElement = XMLElement {
    name :: String,
    attributes :: Map String String,
    children :: [XMLNode]
}
```

### 2.2 XPath查询

**XPath表达式**：

```haskell
-- XPath表达式
data XPath = XPath {
    path :: PathExpression,
    predicate :: Maybe Predicate
}

-- 路径表达式
data PathExpression =
    AbsolutePath [Step]
  | RelativePath [Step]
```

## 4. 实际应用

### 4.1 PostgreSQL XML支持

**XML数据类型**：

```sql
-- 创建XML列
CREATE TABLE documents (
    id INTEGER,
    content XML
);

-- XPath查询
SELECT
    id,
    (xpath('/book/title/text()', content))[1] AS title
FROM documents;

-- XML索引
CREATE INDEX idx_documents_content ON documents
USING GIN (content);
```
