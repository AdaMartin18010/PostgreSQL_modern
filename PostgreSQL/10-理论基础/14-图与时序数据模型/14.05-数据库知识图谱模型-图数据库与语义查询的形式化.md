## 4. 实际应用

### 4.1 知识图谱实现

**使用Apache AGE**：

```sql
-- 创建图
SELECT * FROM cypher('graph', $$
    CREATE (a:Person {name: 'Alice'})
    CREATE (b:Person {name: 'Bob'})
    CREATE (a)-[:KNOWS]->(b)
$$) AS (result agtype);

-- SPARQL查询
SELECT * FROM sparql_query('
    SELECT ?name WHERE {
        ?person :name ?name
    }
');
```
