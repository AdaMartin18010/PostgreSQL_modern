**自动化脚本**:

```bash
# 建议添加到 CI/CD
# .github/workflows/link-check.yml

name: Monthly Link Check
on:
  schedule:
    - cron: '0 0 1 * *'  # 每月1号
  workflow_dispatch:

jobs:
  check-links:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Check links
        run: |
          # 运行链接检查脚本
          python tools/check-links.py
```
