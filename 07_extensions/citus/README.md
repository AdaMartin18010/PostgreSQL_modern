# citus

## 主题边界

- PostgreSQL 的分布式扩展：分片、分布式查询与事务

## 核心要点

- 安装与启用：`CREATE EXTENSION citus;`
- 分片表：`create_distributed_table`、分片键选择
- 查询：分布式执行计划、路由/重分布
- 事务：一致性、分布式事务注意事项
- 运维：节点管理、扩缩容、故障切换

## 知识地图

- 数据建模 → 分片策略 → 查询与事务 → 运维与高可用
