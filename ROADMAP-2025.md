    image: prom/prometheus

  grafana:
    image: grafana/grafana

  pgm-web:
    build: ./web
    ports:
      - "3000:3000"

## 🛠️ 工具开发计划

### 新工具

#### 1. pgm-doctor (诊断工具)

```python
# 一键诊断
pgm doctor --dbname mydb

输出:
✓ 连接正常
⚠️ 缓存命中率偏低 (89%)
✗ 发现3个未使用索引
✓ 无锁等待
⚠️ 2个长事务

建议:
1. 增加shared_buffers到16GB
2. 删除idx_users_old索引
3. 优化慢查询#234
```

#### 2. pgm-optimizer (优化器)

```python
# 自动优化
pgm optimize --dbname mydb --mode=aggressive

执行:
1. VACUUM 5个膨胀表
2. REINDEX 3个索引
3. 更新统计信息
4. 调整配置参数

预计提升:
- 查询速度: +30%
- 存储空间: -15%
```

#### 3. pgm-migrator (迁移工具)

```python
# 智能迁移
pgm migrate \
    --from mysql://localhost/olddb \
    --to postgresql://localhost/newdb \
    --verify

特性:
- 自动类型转换
- 语法兼容转换
- 增量同步
- 数据验证
- 一键回滚
```
