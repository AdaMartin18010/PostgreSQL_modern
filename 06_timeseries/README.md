# 06_timeseries

> 版本对标（更新于 2025-10）

## 主题边界

- PostgreSQL 上的时序数据建模、压缩、连续聚合与保留策略

## 核心要点

- Hypertable 与分区维度设计（时间/空间）
- 压缩与保留策略（TTL/Retention）
- 连续聚合（实时/近实时）
- 写入与查询模式（批写/乱序写/窗口查询）
- 监控与维护（统计、迁移、扩容）

## 知识地图

- 模型 → 写入/查询 → 压缩/聚合 → 维护与扩容
