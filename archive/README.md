# 归档目录说明

本目录用于存放已完成的临时文件、改进补充文档和历史报告。

## 目录结构

- **改进补充文档/**: 已整合到主文档的改进补充文件
  - 这些文件的内容已经整合到对应的主文档中
  - 保留作为历史记录

- **完成报告/**: 各种完成报告、总结报告、进展报告
  - 这些是项目推进过程中的阶段性报告
  - 已过时，保留作为历史记录

- **临时文件/**: 临时生成的文件
  - 代码扫描报告
  - 质量检查报告
  - 会话总结等临时文件

- **根目录归档/**: 从项目根目录迁移进来的“非入口类”历史文件
  - 主要是：阶段性完成/计划/总结/状态/模板/过程跟踪类文档
  - 目录下按类别分桶：`完成报告/`、`计划文档/`、`总结文档/`、`状态文档/`、`模板文档/`、`其他/`
