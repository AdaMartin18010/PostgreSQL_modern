由于工具运行时间较长，建议：

1. **后台运行**:

   ```bash
   python scripts/content_deduplication_checker.py --root docs --threshold 0.7 --output duplication_report.md > dedup_log.txt 2>&1 &
   ```

2. **限制扫描范围**（更快）:

   ```bash
   python scripts/content_deduplication_checker.py --root docs/01-PostgreSQL18 --threshold 0.7 --output duplication_report_pg18.md
   ```

3. **提高阈值**（减少输出）:

   ```bash
   python scripts/content_deduplication_checker.py --root docs --threshold 0.8 --output duplication_report_high.md
   ```
