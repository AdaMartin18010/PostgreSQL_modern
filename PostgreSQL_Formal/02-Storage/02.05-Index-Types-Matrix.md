# Index Types 对比矩阵

> **文档类型**: 存储引擎
> **创建日期**: 2026-03-04

---

## 1. 索引类型对比

| 索引类型 | 数据结构 | 适用操作 | 空间复杂度 | 时间复杂度 | PG扩展 |
|----------|----------|----------|------------|------------|--------|
| **B-tree** | B+树 | =, <, >, <=, >= | O(N) | O(log N) | 内置 |
| **Hash** | 哈希表 | = | O(N) | O(1) | 内置 |
| **GIN** | B+树 + 倒排 | 包含, @> | O(N) | O(log N + M) | 内置 |
| **GiST** | 通用搜索树 | 最近邻, 范围 | O(N) | O(log N) | 内置 |
| **SP-GiST** | 空间分区 | 空间数据 | O(N) | O(log N) | 内置 |
| **BRIN** | 块范围 | 大表范围 | O(N/B) | O(N/B) | 内置 |

## 2. 决策矩阵

### 2.1 选择决策树

```text
[选择哪种索引?]
      |
      +-- [等值查询?]
              |
              +-- [范围查询?]
              |       |
              |       +-- [是] → [B-tree]
              |       |
              |       +-- [否] → [Hash]
              |
              +-- [全文搜索?]
                      |
                      +-- [是] → [GIN]
                      |
                      +-- [空间数据?]
                              |
                              +-- [是] → [GiST/SP-GiST]
                              |
                              +-- [否] → [B-tree]
```
