# PostgreSQL 特性演进时间线

> 📅 最后更新: 2026-04-07
>
> 本时间线按特性类别展示 PostgreSQL 从 PG9.5 到 PG17+ 的功能演进历程。

---

## JSON 支持演进

### 时间线

```
PG9.2 ──► PG9.3 ──► PG9.4 ──► PG9.5 ──► PG12 ──► PG13 ──► PG15 ──► PG16 ──► PG17+
  │        │        │        │        │        │        │        │
  │        │        │        │        │        │        │        └── JSON_TABLE
  │        │        │        │        │        │        └── SQL/JSON 构造器
  │        │        │        │        │        └── JSONB 路径增强
  │        │        │        │        └── JSON 路径表达式
  │        │        │        └── JSONB 路径操作符 (||, -, #>)
  │        │        └── JSONB 数据类型 (二进制存储)
  │        └── JSON 操作函数
  └── 初始 JSON 支持
```

### 各版本详情

| 版本 | JSON 特性 | 说明 |
|------|-----------|------|
| **PG9.2** | 初始 JSON 支持 | `json` 数据类型，文本存储 |
| **PG9.3** | JSON 操作函数 | `json_extract_path()`, `array_to_json()` 等 |
| **PG9.4** | **JSONB 数据类型** | 二进制存储，支持索引，GIN 操作符类 |
| **PG9.5** | JSONB 操作符 | `||` (连接), `-` (删除), `#>` (路径提取) |
| **PG12** | JSON 路径表达式 | SQL/JSON 路径语言支持 |
| **PG13** | JSONB 路径增强 | 更多路径操作函数 |
| **PG15** | SQL/JSON 路径 | 标准化路径表达式支持 |
| **PG16** | **SQL/JSON 构造器** | `JSON_ARRAY()`, `JSON_OBJECT()` 函数 |
| **PG17** | **JSON_TABLE** | 将 JSON 转换为关系表格式 |

## 分区表演进

### 时间线

```
PG8.1 ──► PG10 ──► PG11 ──► PG12 ──► PG13 ──► PG14 ──► PG15 ──► PG16
  │        │        │        │        │        │        │        │
  │        │        │        │        │        │        │        └── 分区查找优化
  │        │        │        │        │        │        └── 多分区性能优化
  │        │        │        │        │        └── DETACH CONCURRENTLY
  │        │        │        │        └── 分区逻辑复制
  │        │        │        └── ATTACH CONCURRENTLY
  │        │        └── 哈希分区, 默认分区, 主键/外键
  │        └── 声明式分区 (RANGE/LIST)
  └── 继承式分区 (触发器方式)
```

### 各版本详情

| 版本 | 分区特性 | 说明 |
|------|----------|------|
| **PG8.1-9.6** | 继承式分区 | 使用表继承 + 触发器手动实现 |
| **PG10** | **声明式分区** | `PARTITION BY RANGE/LIST`，自动路由 |
| **PG11** | **哈希分区** | `PARTITION BY HASH` |
| **PG11** | **默认分区** | 捕获不匹配任何分区的数据 |
| **PG11** | **约束增强** | 分区表支持主键、外键、唯一约束 |
| **PG11** | **分区键更新** | UPDATE 可自动迁移数据到正确分区 |
| **PG12** | **非阻塞 ATTACH** | `ALTER TABLE ATTACH PARTITION` 不阻塞查询 |
| **PG12** | 外键支持 | 支持外键引用分区表 |
| **PG13** | **逻辑复制** | 分区表支持逻辑复制 |
| **PG13** | BEFORE 触发器 | 支持 BEFORE 行级触发器 |
| **PG14** | **非阻塞 DETACH** | `ALTER TABLE DETACH PARTITION CONCURRENTLY` |
| **PG14** | 性能优化 | UPDATE/DELETE 分区表性能提升 |
| **PG15** | 计划生成优化 | 多分区表执行计划生成加速 |
| **PG16** | 分区查找优化 | RANGE/LIST 分区查找增强 |

## 并行查询演进

### 时间线

```
PG9.6 ──► PG10 ──► PG11 ──► PG12 ──► PG13 ──► PG14 ──► PG15
   │        │        │        │        │        │        │
   │        │        │        │        │        │        └── SELECT DISTINCT 并行
   │        │        │        │        │        └── 并行优化持续
   │        │        │        │        └── 并行 VACUUM
   │        │        │        └── 可串行化并行
   │        │        └── 并行哈希连接, 并行 CREATE INDEX
   │        └── 并行索引扫描, 并行合并连接
   └── 并行顺序扫描, 并行聚合
```

### 各版本详情

| 版本 | 并行特性 | 说明 |
|------|----------|------|
| **PG9.6** | **并行顺序扫描** | `max_parallel_workers_per_gather` 控制 |
| **PG9.6** | **并行聚合** | 聚合操作并行化 |
| **PG10** | **并行索引扫描** | 位图索引扫描并行化 |
| **PG10** | **并行合并连接** | Merge Join 并行支持 |
| **PG10** | 并行分区扫描 | 分区表并行扫描 |
| **PG11** | **并行哈希连接** | Hash Join 完全并行支持 |
| **PG11** | **并行 CREATE INDEX** | B-tree 索引并行创建 |
| **PG12** | 可串行化并行 | 可串行化隔离级别支持并行查询 |
| **PG12** | JIT 编译 | LLVM JIT 加速查询执行 |
| **PG13** | 并行 VACUUM | 并行处理索引 |
| **PG15** | **SELECT DISTINCT 并行** | DISTINCT 操作并行化 |

## 索引类型演进

### 时间线

```
PG8.2 ──► PG8.3 ──► PG9.2 ──► PG9.4 ──► PG9.5 ──► PG9.6 ──► PG12
   │        │        │        │        │        │        │
   │        │        │        │        │        │        └── REINDEX CONCURRENTLY
   │        │        │        │        │        └── 并行 CREATE INDEX
   │        │        │        │        └── BRIN 索引
   │        │        │        └── GIN 索引增强
   │        │        └── SP-GiST 索引
   │        └── GIN 全文索引
   └── GiST 通用索引
```

### 索引类型支持矩阵

| 索引类型 | 引入版本 | 关键增强 |
|----------|----------|----------|
| **B-tree** | 最初 | PG11: 并行创建, PG12: 优化 |
| **GiST** | PG8.2 | 空间数据、范围类型 |
| **GIN** | PG8.3 | PG9.4: JSONB 支持 |
| **SP-GiST** | PG9.2 | 空间分区 GiST |
| **BRIN** | PG9.5 | 块范围索引，适合时序数据 |
| **Hash** | PG10 | PG10: 事务安全，可崩溃恢复 |

   │        │        │        │        │        │        │        │        │
   │        │        │        │        │        │        │        │        └── security_invoker
   │        │        │        │        │        │        │        └── SCRAM-SHA-256
   │        │        │        │        │        │        └── 默认权限改进
   │        │        │        │        │        └── 行级安全策略
   │        │        │        │        └── 角色继承增强
   │        │        │        └── 行级安全 (RLS)
   │        │        └── 列级加密扩展
   │        └── 扩展权限控制

### 各版本详情

| 版本 | 安全特性 | 说明 |
|------|----------|------|

| **PG13** | SCRAM 增强 | 默认认证方式 |

## 性能优化演进

### 时间线

```
PG9.2 ──► PG9.6 ──► PG10 ──► PG11 ──► PG12 ──► PG13 ──► PG15
   │        │        │        │        │        │        │
   │        │        │        │        │        │        └── 排序优化 (400%)
   │        │        │        │        │        └── 增量排序
   │        │        │        │        └── JIT 编译
   │        │        │        └── 并行优化
   │        │        └── 多核扩展
   │        └── 并行查询引入
   └── 索引仅扫描
```

### 关键性能里程碑

| 版本 | 性能特性 | 典型提升 |
|------|----------|----------|
| **PG9.2** | 索引仅扫描 | 减少随机 I/O |
| **PG9.6** | 并行查询 | 多核 CPU 利用 |
| **PG10** | 连接优化 | 哈希连接、合并连接并行 |
| **PG11** | 并行索引创建 | 大索引创建加速 |
| **PG12** | **JIT 编译** | 复杂查询 10-30% 提升 |
| **PG12** | CTE 内联 | 避免物化，性能提升 |
| **PG13** | **增量排序** | 避免重复排序 |
| **PG15** | **排序优化** | 25%-400% 性能提升 |
| **PG15** | 压缩增强 | WAL/备份压缩减少 I/O |
