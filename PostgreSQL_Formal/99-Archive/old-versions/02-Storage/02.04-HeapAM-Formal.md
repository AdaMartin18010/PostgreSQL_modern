# Heap Access Method 形式化

> **文档类型**: 存储引擎
> **创建日期**: 2026-03-04

---

## 1. 概念定义

### 1.1 堆表结构

表作为页的集合:

$$
\text{Table} = \{P_1, P_2, ..., P_n\}
$$

每页包含元组:

$$
P_i = \{t_1, t_2, ..., t_m\}
$$

### 1.2 元组格式

```
元组头 (HeapTupleHeader):
┌─────────┬─────────┬─────────┬─────────┐
│ t_xmin  │ t_xmax  │ t_cid   │ 其他    │
└─────────┴─────────┴─────────┴─────────┘
```

---

## 2. 操作

### 2.1 插入

```
1. 找到有空闲空间的页
2. 写入元组数据
3. 设置t_xmin = 当前事务ID
4. 更新页空闲空间映射
```

### 2.2 更新 (MVCC)

```
1. 插入新版本元组 (t_xmin = 新事务ID)
2. 旧版本标记为删除 (t_xmax = 新事务ID)
3. 更新索引指向新版本
```

### 2.3 删除

```
1. 设置t_xmax = 当前事务ID
2. 元组对后续事务不可见
```
