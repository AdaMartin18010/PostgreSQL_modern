## 3. 模型检查配置

### 3.1 TLC配置

```tla
-- MVCC.cfg
CONSTANTS
    Transactions = {T1, T2, T3}
    Objects = {x, y}

INIT Init
NEXT Next

INVARIANTS
    TypeInvariant
    Consistency

PROPERTIES
    Serializability
```

### 3.2 验证命令

```bash
# 运行TLC模型检查
tlc MVCC.tla -config MVCC.cfg

# 生成状态图
tlc MVCC.tla -dump dot,actionlabels states.dot
```
