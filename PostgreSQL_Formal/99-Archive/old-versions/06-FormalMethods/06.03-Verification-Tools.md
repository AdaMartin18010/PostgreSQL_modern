## 2. CI/CD集成

```yaml
# .github/workflows/verify.yml
name: Formal Verification
on: [push]
jobs:
  tla:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Run TLC
        run: tlc MVCC.tla
```
