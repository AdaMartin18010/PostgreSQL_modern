# Virtual Generated Columns 形式化分析

> **文档类型**: PostgreSQL 18新特性
> **创建日期**: 2026-03-04

---

## 1. 概念定义

### 1.1 生成列类型

**Stored Generated Column**:
$$
\text{col}_{stored} = f(c_1, c_2, ...) \quad \text{(物理存储)}
$$

**Virtual Generated Column** (PG18默认):
$$
\text{col}_{virtual} = f(c_1, c_2, ...) \quad \text{(计算时生成)}
$$

### 1.2 语义差异

| 特性 | STORED | VIRTUAL |
|------|--------|---------|
| 存储 | 是 | 否 |
| 读取性能 | 快 | 需计算 |
| 写入性能 | 慢(需计算) | 快 |
| 空间占用 | 有 | 无 |

---

## 2. 形式化语义

**惰性计算**:
$$
\text{value}(col, t) =
\begin{cases}
    \text{compute}(f, t) & \text{if accessed} \\
    \bot & \text{otherwise}
\end{cases}
$$

---

## 3. PostgreSQL实现

```sql
-- 创建虚拟生成列
CREATE TABLE products (
    id SERIAL PRIMARY KEY,
    price NUMERIC,
    quantity INT,
    total_value NUMERIC GENERATED ALWAYS AS (price * quantity) VIRTUAL
);

-- 逻辑复制支持
-- PG18: 生成列可复制到订阅端
```
