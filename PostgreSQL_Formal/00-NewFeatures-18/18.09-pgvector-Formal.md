## 1. 概念定义

### 1.1 向量空间

$$
\mathbb{R}^d := \{v = (v_1, v_2, ..., v_d) \mid v_i \in \mathbb{R}\}
$$

### 1.2 距离度量

| 度量 | 公式 | 适用场景 |
|------|------|----------|
| **欧氏距离** | $L_2(u,v) = \sqrt{\sum_{i=1}^{d}(u_i-v_i)^2}$ | 连续数据 |
| **余弦相似度** | $\cos(u,v) = \frac{u \cdot v}{\|u\| \|v\|}$ | 方向重要 |
| **曼哈顿距离** | $L_1(u,v) = \sum_{i=1}^{d}|u_i-v_i|$ | 稀疏数据 |
| **汉明距离** | $H(u,v) = \sum_{i=1}^{d}\mathbb{1}_{u_i \neq v_i}$ | 二值数据 |

## 2. 索引算法

### 2.1 HNSW (Hierarchical Navigable Small World)

**图结构**:

- 多层图结构
- 每层是proximity graph
- 搜索时从顶层开始

**复杂度**:

- 构建: $O(n \log n)$
- 搜索: $O(\log n)$

### 2.2 IVFFlat

**倒排文件索引**:

1. K-means聚类
2. 分配到最近质心
3. 搜索时只查相关列表

## 3. PostgreSQL实现

```sql
-- 安装pgvector
CREATE EXTENSION vector;

-- 创建向量列
CREATE TABLE items (
    id BIGINT PRIMARY KEY,
    embedding VECTOR(1536)
);

-- 创建HNSW索引
CREATE INDEX ON items USING hnsw (embedding vector_l2_ops);

-- 近似最近邻查询
SELECT * FROM items
ORDER BY embedding <-> '[1,2,3,...]'
LIMIT 10;
```
