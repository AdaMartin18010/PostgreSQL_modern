> **创建日期**: 2026-03-04

---

## 1. 并行构建算法

### 1.1 分阶段构建

```
Phase 1: 并行扫描表
    - 多个worker并行读取堆数据
    - 各自构建本地 postings list

Phase 2: 合并
    - 合并各worker的 postings
    - 构建最终B-tree

Phase 3: 写入
    - 写入索引页
```
