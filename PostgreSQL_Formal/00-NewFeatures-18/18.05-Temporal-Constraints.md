# Temporal Constraints 形式化分析

> **文档类型**: PostgreSQL 18新特性
> **创建日期**: 2026-03-04

---

## 1. 概念定义

### 1.1 时态约束

**WITHOUT OVERLAPS**: 确保时间段不重叠

$$
\forall t_1, t_2 \in R: [t_1.start, t_1.end) \cap [t_2.start, t_2.end) = \emptyset
$$

**PERIOD**: 定义有效时间段

---

## 2. 形式化语义

### 2.1 区间代数

| 关系 | 定义 |
|------|------|
| Before | $end_1 < start_2$ |
| Meets | $end_1 = start_2$ |
| Overlaps | $start_1 < start_2 < end_1 < end_2$ |
| Contains | $start_1 < start_2 \land end_2 < end_1$ |

---

## 3. PostgreSQL实现

```sql
CREATE TABLE reservations (
    room_id INT,
    guest_name TEXT,
    during TSTZRANGE,
    EXCLUDE USING gist (room_id WITH =, during WITH &&)
);

-- PG18简化语法:
CREATE TABLE reservations (
    room_id INT,
    guest_name TEXT,
    valid_from TIMESTAMPTZ,
    valid_to TIMESTAMPTZ,
    PERIOD FOR valid_period (valid_from, valid_to),
    CONSTRAINT no_overlap UNIQUE (room_id, valid_period WITHOUT OVERLAPS)
);
```
