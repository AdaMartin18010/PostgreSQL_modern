# pg_upgrade Enhancements 形式化

> **文档类型**: PostgreSQL 18新特性
> **创建日期**: 2026-03-04

---

## 1. 统计信息保留

PG18允许升级时保留优化器统计信息:

```bash
pg_upgrade --retain-statistics
```

避免升级后性能下降。

---

## 2. 并行升级

```bash
pg_upgrade --jobs=8
```

加速大集群升级。
