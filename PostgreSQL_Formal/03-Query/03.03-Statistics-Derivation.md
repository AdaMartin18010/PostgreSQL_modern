## 1. 统计信息类型

### 1.1 基本统计

| 统计项 | 符号 | 说明 |
|--------|------|------|
| 行数 | $n$ | 表总行数 |
| 不同值 | $d$ | 列不同值数 |
| 空值比例 | $null\_frac$ | NULL值比例 |
| 平均宽度 | $avg\_width$ | 平均字节数 |

### 1.2 直方图

等深直方图:
$$
bucket_i = [v_{i}, v_{i+1}), \quad |bucket_i| \approx \frac{n}{k}
$$

## 2. 选择率推导

### 2.1 等值条件

```
sel(A = v) = 1 / d(A)   (如果v在MCV中)
sel(A = v) = (1 - sum(MCV)) / (d - n_MCV)   (否则)
```

### 2.2 范围条件

```
sel(A < v) = cumulative_histogram(v)
```
