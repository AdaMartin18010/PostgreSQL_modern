## 1. JIT原理

### 1.1 执行流程

```
查询计划
    ↓
Expression Tree
    ↓
LLVM IR Generation
    ↓
LLVM Optimization
    ↓
Machine Code
    ↓
Direct Execution
```

### 1.2 加速场景

| 场景 | 加速比 | 原因 |
|------|--------|------|
| 复杂表达式 | 2-5x | 消除解释开销 |
| 大批量处理 | 1.5-3x | SIMD优化 |
| 重复执行 | 5-10x | 代码缓存 |

---

## 2. PostgreSQL配置

```ini
jit = on
jit_above_cost = 100000
jit_inline_above_cost = 500000
jit_optimize_above_cost = 500000
```
