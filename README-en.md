## Goal

PostgreSQL_Modern is a comprehensive, Chinese-language knowledge base for PostgreSQL 18.1. It covers core features, performance tuning, AI/ML integration (pgvector, RAG), distributed systems, production operations, and theory. Content is aligned with official PostgreSQL documentation and international best practices.
