## 📋 概述

本思维导图展示可串行化定理证明的完整结构，帮助理解可串行化判定、保证和冲突可串行化的证明过程。

**核心观点**：

- **可串行化判定定理**：调度可串行化当且仅当存在等价串行顺序
- **可串行化保证定理**：SERIALIZABLE隔离级别保证可串行化
- **冲突可串行化定理**：冲突可串行化是可串行化的充分条件

---

## 📊 第一部分：可串行化定理结构

```text
可串行化定理
│
├─ 可串行化判定定理
│   ├─ 定义：serializable(S) ⟺ ∃serial_order, result(S) = result(serial_order)
│   ├─ 必要性：serializable(S) ⟹ ∃serial_order
│   └─ 充分性：∃serial_order ⟹ serializable(S)
│
├─ 可串行化保证定理
│   ├─ 前提：isolation_level(τ) = SERIALIZABLE
│   ├─ 结论：∀schedule S containing τ, serializable(S)
│   └─ 机制：SERIALIZABLE隔离级别保证可串行化
│
└─ 冲突可串行化定理
    ├─ 定义：conflict_serializable(S)
    ├─ 结论：conflict_serializable(S) ⟹ serializable(S)
    └─ 关系：冲突可串行化 ⟹ 可串行化
```

## 📊 第三部分：冲突可串行化

```text
冲突可串行化
│
├─ 冲突定义
│   ├─ 写-读冲突：write(τ₁, r) → read(τ₂, r)
│   ├─ 读-写冲突：read(τ₁, r) → write(τ₂, r)
│   └─ 写-写冲突：write(τ₁, r) → write(τ₂, r)
│
├─ 冲突图
│   ├─ 节点：事务
│   ├─ 边：冲突关系
│   └─ 无环性：冲突可串行化 ⟺ 冲突图无环
│
└─ 冲突可串行化判定
    ├─ 方法：构建冲突图
    ├─ 判定：检查是否有环
    └─ 结论：无环 ⟹ 冲突可串行化
```
