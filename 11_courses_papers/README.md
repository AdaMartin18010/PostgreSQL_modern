# 11_courses_papers

> 版本对标（更新于 2025-09）

## 目标

- 建立“高校课程/教材/论文”的权威索引与主题映射，支撑各目录的对标与延伸阅读。

## 对标框架

- 主题 → 课程/教材 → 核心讲义/实验 → 参考论文
- 优先级：官方/高校 → 权威出版 → 顶会/顶刊 → 社区精选

- 数据库系统课程（本科/研究生）
