**为什么使用简短命名（mvcc00.md等）**：

- 这些文档已经广泛引用，重命名影响较大
- 通过文档编号和README可以快速识别文档主题
- 保持命名的简洁性

**如何快速识别文档**：

- 查看文档编号（MVCC-001, CAP-001等）
