## 📋 概述

本思维导图展示MVCC核心公理系统的完整结构，帮助理解MVCC的形式化数学基础。

**核心观点**：

- **形式化基础**：符号定义、基本概念
- **核心公理**：版本公理、可见性公理、快照公理、版本链公理
- **推理规则**：版本推理、可见性推理、快照推理
- **公理一致性**：独立性、完备性、一致性证明

## 📊 第一部分：形式化基础

```text
形式化基础
│
├─ 符号定义
│   ├─ 基本符号
│   │   ├─ T：事务集合
│   │   ├─ V：版本集合
│   │   ├─ R：元组集合
│   │   └─ S：快照集合
│   │
│   └─ 关系符号
│       ├─ xmin(v)：版本创建事务ID
│       ├─ xmax(v)：版本删除事务ID
│       ├─ snapshot(τ)：事务快照
│       ├─ visible(v, s)：版本可见性
│       ├─ committed(τ)：事务已提交
│       └─ aborted(τ)：事务已中止
│
└─ 基本概念
    ├─ 版本：v = (xmin, xmax, data, timestamp)
    ├─ 快照：s = {τ₁, τ₂, ..., τₙ}
    └─ 版本链：C(r) = [v₁, v₂, ..., vₙ]
```
