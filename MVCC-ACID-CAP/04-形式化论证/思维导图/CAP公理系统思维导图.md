## 📋 概述

本思维导图展示CAP公理系统的完整结构，帮助理解CAP定理的形式化数学基础。

**核心观点**：

- **形式化基础**：符号定义、基本概念
- **CAP公理**：一致性、可用性、分区容错公理
- **权衡公理**：CAP三元权衡的不可能性
- **推理规则**：从公理推导CAP性质

## 📊 第一部分：形式化基础

```text
形式化基础
│
├─ 符号定义
│   ├─ 基本符号
│   │   ├─ S：分布式系统
│   │   ├─ N：节点集合
│   │   ├─ P：分区集合
│   │   ├─ O：操作集合
│   │   ├─ R：读取操作集合
│   │   └─ W：写入操作集合
│   │
│   └─ 关系符号
│       ├─ consistent(S)：系统一致性
│       ├─ available(S)：系统可用性
│       ├─ partition_tolerant(S)：分区容错性
│       ├─ read(n, k)：节点读取
│       └─ write(n, k, v)：节点写入
│
└─ 基本概念
    ├─ 分布式系统：S = (N, O)
    ├─ 分区：p = {N₁, N₂, ..., Nₖ}
    ├─ 一致性：∀n₁, n₂, read(n₁, k) = read(n₂, k)
    ├─ 可用性：∀n, ∀o, responds(n, o)
    └─ 分区容错性：system_continues_operating(S, p)
```
