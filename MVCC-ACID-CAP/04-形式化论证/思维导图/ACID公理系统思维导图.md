## 📋 概述

本思维导图展示ACID公理系统的完整结构，帮助理解ACID属性的形式化数学基础。

**核心观点**：

- **形式化基础**：符号定义、基本概念
- **ACID公理**：原子性、一致性、隔离性、持久性公理
- **关系公理**：ACID属性之间的相互关系
- **推理规则**：从公理推导ACID性质

---

## 📊 第一部分：形式化基础

```text
形式化基础
│
├─ 符号定义
│   ├─ 基本符号
│   │   ├─ T：事务集合
│   │   ├─ O(τ)：事务操作集合
│   │   ├─ S：数据库状态集合
│   │   └─ C：一致性约束集合
│   │
│   └─ 关系符号
│       ├─ begin(τ)：事务开始
│       ├─ commit(τ)：事务提交
│       ├─ abort(τ)：事务中止
│       ├─ state(τ)：事务状态
│       └─ satisfies(s, c)：状态满足约束
│
└─ 基本概念
    ├─ 事务：τ = [o₁, o₂, ..., oₙ]
    ├─ 数据库状态：s: R → V
    └─ 一致性约束：c: S → {true, false}
```
