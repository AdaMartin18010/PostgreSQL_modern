## 📊 第四部分：思维导图格式规范

### 格式要求

1. **文档结构**：
   - 文档编号：MINDMAP-XXX
   - 主题：对应原文档主题
   - 版本：PostgreSQL 17 & 18
   - 状态：✅ 已完成 / 🔄 进行中 / ⏳ 待创建

2. **内容结构**：
   - 📑 目录
   - 📋 概述
   - 📊 第一部分：核心概念
   - 📊 第二部分：关系结构
   - 📊 第三部分：实践应用
   - 📚 相关文档链接

3. **文本树形结构**：
   ```text
   核心概念
   │
   ├─ 子概念1
   │   ├─ 子子概念1.1
   │   └─ 子子概念1.2
   └─ 子概念2
   ```

4. **关联关系**：
   - 使用箭头 `→` 表示概念间的关联
   - 使用 `↔` 表示双向关系
   - 使用 `↗` `↘` 表示层次关系
