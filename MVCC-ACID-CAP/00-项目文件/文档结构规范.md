### 2. 主题编号规范

#### 一级主题（##）

- 格式：`## 📊 第一部分：主题名称`
- 编号：第一部分、第二部分、第三部分...
- 图标：📊 🔍 🚀 📈 🎯 🔧 📝 等

#### 二级主题（###）

- 格式：`### 1.1 子主题名称`
- 编号：1.1、1.2、2.1、2.2...
- 不使用图标

#### 三级主题（####）

- 格式：`#### 具体内容标题`
- 不使用编号，直接使用描述性标题

### 4. 目录生成规则

#### 自动生成目录

- 所有一级主题（##）必须包含在目录中
- 重要二级主题（###）建议包含在目录中
- 目录链接使用Markdown锚点格式

#### 目录格式示例

```markdown
## 📑 目录

- [概述](#概述)
- [第一部分：标题](#第一部分标题)
  - [1.1 子标题](#11-子标题)
  - [1.2 子标题](#12-子标题)
- [第二部分：标题](#第二部分标题)
- [总结](#总结)
```

### 5. 章节编号一致性

#### 必须保持一致

- 同一文档内的章节编号必须连续
- 跨文档的同类主题编号应保持一致
- 子章节编号格式：X.Y（X为父章节，Y为子章节序号）

#### 编号示例

```markdown
## 📊 第一部分：基础概念
### 1.1 概念A
### 1.2 概念B

## 🚀 第二部分：实现机制
### 2.1 机制A
### 2.2 机制B
```
