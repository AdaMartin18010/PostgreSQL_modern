## 📊 第一部分：依赖关系概述

### 1.1 依赖关系定义

文档依赖关系表示文档之间的引用、基于、扩展等关系。

### 1.2 依赖类型

1. **基础依赖**：A基于B（如：mvcc02.md基于mvcc00.md）
2. **引用依赖**：A引用B（如：文档A引用文档B的定义）
3. **扩展依赖**：A扩展B（如：mvcc04.md扩展mvcc03.md）
4. **验证依赖**：A验证B（如：场景文档验证理论文档）
5. **应用依赖**：A应用B（如：实践文档应用理论文档）

### 1.3 依赖层次

```text
文档依赖层次
│
├─ 核心层（view/）
│   └─ 基础文档，被其他文档依赖
│
├─ 理论层（01-理论基础/）
│   ├─ 依赖核心层
│   └─ 被其他层依赖
│
├─ 视角层（02-多维度视角/）
│   ├─ 依赖核心层和理论层
│   └─ 被实践层依赖
│
├─ 实践层（03-场景实践/）
│   ├─ 依赖核心层、理论层、视角层
│   └─ 被验证层依赖
│
└─ 论证层（04-形式化论证/）
    ├─ 依赖核心层和理论层
    └─ 验证其他层
```

## 📊 第七部分：依赖类型定义

### 7.1 基础依赖 (based_on)

**定义**：A基于B表示A以B为基础建立。

**符号**：A →based_on B

**示例**：

- mvcc01.md →based_on mvcc00.md
- mvcc02.md →based_on mvcc00.md
- mvcc03.md →based_on mvcc00.md

**强度**：高（基础依赖）

### 7.2 引用依赖 (references)

**定义**：A引用B表示A引用B的内容。

**符号**：A →references B

**示例**：

- mvcc04.md →references mvcc01.md
- ACID属性定理证明.md →references ACID公理系统.md
- 场景实践文档 →references 理论文档

**强度**：中（内容引用）

### 7.3 扩展依赖 (extends)

**定义**：A扩展B表示A在B的基础上扩展。

**符号**：A →extends B

**示例**：

- mvcc03.md →extends mvcc02.md
- mvcc04.md →extends mvcc03.md
- mvcc05.md →extends mvcc04.md

**强度**：高（功能扩展）

### 7.4 验证依赖 (verifies)

**定义**：A验证B表示A验证B的正确性。

**符号**：A →verifies B

**示例**：

- 场景实践文档 →verifies 理论文档
- 形式化证明文档 →verifies 公理系统
- 测试工具 →verifies 实现文档

**强度**：中（验证关系）

### 7.5 应用依赖 (applies)

**定义**：A应用B表示A应用B的理论或方法。

**符号**：A →applies B

**示例**：

- 实践文档 →applies 理论文档
- 配置文档 →applies 理论文档
- 优化文档 →applies 理论文档

**强度**：中（应用关系）
