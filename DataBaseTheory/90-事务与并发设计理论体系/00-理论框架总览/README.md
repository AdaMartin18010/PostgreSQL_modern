- $\land, \lor, \neg, \implies, \iff$
- $\forall, \exists$

**第三部分: 关系符号**

- $\prec$ (偏序)

- $\equiv$ (等价)

**第四部分: 自定义符号**

- $Visible(s, t, obs)$ (可见性谓词)
