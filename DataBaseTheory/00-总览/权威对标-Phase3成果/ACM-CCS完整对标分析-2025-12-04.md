```text
Information systems
└── Data management (数据管理)
    ├── Database management system engines (数据库管理系统引擎)
    │   ├── Relational database systems (关系数据库系统)
    │   ├── Object oriented database systems (面向对象数据库)
    │   ├── Parallel database systems (并行数据库系统)
    │   ├── Distributed database systems (分布式数据库系统)
    │   └── Cloud database systems (云数据库系统)
    │
    ├── Database design and models (数据库设计与模型)
    │   ├── Relational database model (关系数据库模型)
    │   ├── Entity relationship models (实体关系模型)
    │   ├── Graph-based database models (基于图的数据库模型)
    │   └── Physical data models (物理数据模型)
    │
    ├── Query languages (查询语言)
    │   ├── Relational database query languages (关系数据库查询语言)
    │   ├── SQL (SQL)
    │   ├── Query optimization (查询优化)
    │   └── Query operators (查询算子)
    │
    ├── Database transaction processing (数据库事务处理)
    │   ├── Transaction processing systems (事务处理系统)
    │   ├── Database transaction management (数据库事务管理)
    │   ├── Database recovery (数据库恢复)
    │   ├── Concurrency control (并发控制)
    │   └── Distributed transactions (分布式事务)
    │
    ├── Data structures (数据结构)
    │   ├── Data access methods (数据访问方法)
    │   ├── Point lookups (点查找)
    │   ├── Multidimensional range search (多维范围搜索)
    │   └── Nearest-neighbor search (最近邻搜索)
    │
    └── Information systems applications (信息系统应用)
        ├── Data mining (数据挖掘)
        ├── Data analytics (数据分析)
        └── Decision support systems (决策支持系统)
```
