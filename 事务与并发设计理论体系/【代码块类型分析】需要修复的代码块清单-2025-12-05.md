## 📋 代码块类型判断规则

### 判断标准

1. **文本/树状结构** (`text`):
   - 使用 `├─`, `└─`, `│` 等树状符号
   - 纯文本描述，没有代码语法
   - ASCII艺术图

2. **SQL代码** (`sql`):
   - 包含 SQL 关键字: SELECT, UPDATE, INSERT, BEGIN, COMMIT 等
   - PostgreSQL 语法

3. **Python代码** (`python`):
   - Python 语法: def, class, import 等
   - 缩进风格

4. **Rust代码** (`rust`):
   - Rust 语法: fn, let, struct, impl 等
   - 生命周期标记 `'a`

5. **C代码** (`c`):
   - C 语法: struct, typedef, #include 等

6. **Mermaid图表** (`mermaid`):
   - graph, flowchart, sequenceDiagram 等关键字
